## 学生平时成绩维护系统

- Java Http服务器：`Apache Tomcat`
- 数据库：`MySQL`
- 框架：`MVC`

> 能够实现的功能：登录验证、添加学生信息、学生信息维护、录入作业信息、作业信息维护、数据删除、密码修改...
